# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-11

- Prima release.
- Piattaforma Homebridge con multi-webhook, token, sicurezza local-only, debounce/durata.
- Trigger Email via IMAP (Gmail supportato) usando imapflow.
- Endpoint admin per reveal una sola volta e regenerate dei token.
- Schema Homebridge, README e metadati package aggiornati.
